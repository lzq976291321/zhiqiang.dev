Code review Skill 不应该只检查格式，而应该优先找会影响线上行为的问题。

建议检查顺序：

- 用户行为是否改变
- 边界条件是否覆盖
- 错误处理是否可恢复
- 是否遗漏测试
- 是否引入权限或数据风险
- 是否有不必要的复杂抽象

适合在提交前作为最后一轮验证。
