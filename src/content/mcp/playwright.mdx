Playwright MCP 适合做视觉回归、表单流程、控制台错误检查、移动端布局验证和截图确认。

它不是普通爬虫工具。对前端来说，它更像一个能被 Agent 调用的浏览器 QA 层。

注意：浏览器自动化会消耗较多上下文和时间，复杂任务要限制目标页面和验证范围。
