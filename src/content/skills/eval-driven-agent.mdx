Agent 上线前必须有评测闭环。

这个 Skill 会把模糊体验拆成：

- 典型任务集
- 失败类型
- 成功判定
- 工具调用轨迹
- 安全边界检查
- 回归对比方式

它特别适合和 Verification、Permission、Context 设计一起使用。
