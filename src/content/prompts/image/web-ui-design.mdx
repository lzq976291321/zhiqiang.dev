网页着陆页和 UI 界面设计，适合设计参考、竞品分析、快速出效果图。

## 常用风格关键词

- **极简主义**：大留白、单色系、衬线标题
- **毛玻璃**：半透明卡片、模糊背景、渐变色
- **新拟态**：柔和阴影、凸起/凹陷效果
- **杂志风**：网格布局、衬线体、黑白为主
- **暗黑科技**：深色背景、霓虹强调色、代码感
